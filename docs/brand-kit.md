# Qwizzy Brand Kit

## Brand Essence
- Purpose: make quiz battles fast, social, and satisfying.
- Personality: clever, energetic, friendly, competitive (but fair).
- Promise: quick rounds, clear scores, instant bragging rights.

## Logo/Mark Concept
- Primary mark: a bold, rounded “Q” with a speech-bubble tail (quiz + chat + duel).
- Secondary mark: the “Q” inside a rounded square for app icons.
- Clear space: keep at least 1/2 the “Q” height around the mark.

## Color System
- Primary: Qwizzy Orange `#FF7A3D`
- Dark: Ink Navy `#101828`
- Accent: Mint `#23C6A1`
- Neutral Light: Mist `#F6F7FB`
- Neutral Mid: Cloud `#E4E7EC`
- Neutral Dark: Slate `#344054`

Usage guidance:
- Primary for CTAs, highlights, scores.
- Dark for text and UI foundations.
- Accent for success, streaks, and badges.

## Typography
- Headings: Sora (700–800)
- Body: Manrope (400–600)
- Fallbacks: System / San Francisco / Roboto

## UI Style
- Rounded corners (10–16px), strong contrast, bright CTA.
- Scoreboards use bold numerals and color chips.
- Gradients are subtle (Orange -> Coral), never neon.

## Imagery & Illustration
- Minimal, crisp, high-contrast.
- Subtle grid or dot texture backgrounds.
- Avoid busy, cluttered scenes.

## Tone of Voice
- Short, energetic phrases.
- Use friendly competitiveness: “Ready for the next round?”
- Avoid long paragraphs in UI copy.

## App Icon Guidance
- Simple, bold, one-letter mark.
- No thin strokes, no small text.
- High contrast for legibility at 60px.
